# VALORANT Battlepass Tracker

Script to fetch the game client API (unofficial doc [here](https://valapidocs.techchrism.me/)) to track the ingame
battlepass progress.

## Getting Started

Run the script with:

```bash
deno task start
```

Create executable with:

```bash
deno task compile
```
